# 🌿 LeafyAI – Smart Vegan Nutrition Assistant

LeafyAI is an AI-powered vegan nutrition and recipe recommendation platform built using React, FastAPI, and MySQL.

---

## ✨ Features
- Personalized vegan meal recommendations
- Weight loss / muscle gain / maintenance goals
- 50+ vegan recipes
- Beginner-friendly cooking steps
- Detailed nutrition facts
- AI recipe generator using ingredients
- Recipe search & filtering
- Save favorite recipes
- Modern responsive UI

---

## 🛠 Tech Stack

### Frontend
- React.js
- Tailwind CSS
- React Router

### Backend
- FastAPI
- Python

### Database
- MySQL

---

## 🚀 Run Frontend

```bash
cd leafyai
npm install
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```

---

## 🚀 Run Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

Backend runs on:

```bash
http://127.0.0.1:8000
```

---

## 🗄 Database Setup

1. Open MySQL Workbench
2. Create database:

```sql
CREATE DATABASE leafyai;
```

3. Update MySQL credentials in backend config file.
