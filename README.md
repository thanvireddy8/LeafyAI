# 🌿 LeafyAI – Smart Vegan Nutrition Assistant

LeafyAI is an AI-powered vegan nutrition and recipe recommendation platform built using React, FastAPI, and MySQL.

## ✨ Features
- Personalized vegan meal recommendations
- Weight loss / muscle gain / maintenance goals
- 50+ vegan recipes
- Beginner-friendly cooking steps
- Detailed nutrition facts
- AI recipe generator using ingredients
- Recipe search & filtering
- Save favorite recipes
- Modern responsive UI

## 🛠 Tech Stack
### Frontend
- React.js
- Tailwind CSS
- React Router

### Backend
- FastAPI
- Python

### Database
- MySQL

## 📸 Features Included
- Breakfast, Lunch, Dinner, Snacks, Refreshers
- Vegan substitutions
- Nutrition tracking
- AI-powered recipe generation
